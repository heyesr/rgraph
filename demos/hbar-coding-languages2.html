Popular coding languages (an alternative method)

[No canvas support]

This Horizontal Bar chart is a duplicate of the hbar-coding-languages.html public demo but done a different way - instead of using custom coding it uses a second HBar chart.

This goes in the documents header: 
<script src="RGraph.common.core.js"></script>
<script src="RGraph.common.dynamic.js"></script>
<script src="RGraph.common.tooltips.js"></script>
<script src="RGraph.hbar.js"></script>
Put this where you want the chart to show up: 
<canvas id="cvs" width="600" height="350">[No canvas support]</canvas>
