A basic demo of the standalone CSV reader

Note: For browser security reasons the ajax demos don't work offline. You can view the demos on the RGraph website here: https://www.rgraph.net/demos/index.html

This demo doesn't show a chart - but instead just uses the now standalone csv reader to fetch a csv file and show it. Both the canvas and svg versions of the csv reader are standalone. View the source to see the code (CTRL+U in Chrome).
