An SVG stacked Rose chart with tooltips

A Rose chart that has a custom tooltip event listener. The defaults for tooltips have been set and the custom tooltip event sets the appropriate border color.

This goes in the documents header: 
<script src="RGraph.svg.common.core.js"></script>
<script src="RGraph.svg.common.tooltips.js"></script>
<script src="RGraph.svg.rose.js"></script>
Put this where you want the chart to show up: 
<div style="width: 500px; height: 500px" id="chart-container"></div>
