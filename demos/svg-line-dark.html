An SVG Line chart with a dark theme

A dark themed filled Line chart with a red tinge to it. The chart is using a gradient as the fill color and is also using the spline option. The background of the chart is dark by using the colorsBackground option.

This goes in the documents header: 
<script src="../libraries/RGraph.svg.common.core.js" ></script>
<script src="../libraries/RGraph.svg.line.js" ></script>
Put this where you want the chart to show up: 
<div style="width: 950px; height: 250px" id="chart-container"></div>
