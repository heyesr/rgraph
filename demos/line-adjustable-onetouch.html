A Line chart that uses the growPoint() method to facilitate one-touch adjusting

[No canvas support]

This Line chart uses custom adjusting that should work well with touch-based devices (eg tablets) - which the normal Line chart adjusting doesn't.

This goes in the documents header: 
<script src="RGraph.common.core.js"></script>
<script src="RGraph.line.js"></script>
Put this where you want the chart to show up: 

<canvas id="cvs" width="750" height="250">[No canvas support]</canvas>
<button style="font-size: 20pt; cursor: pointer" onclick="line.set('spline', !line.get('spline')); RGraph.redraw();">Toggle spline</button>
