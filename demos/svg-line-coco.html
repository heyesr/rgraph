An SVG Line chart using the trace() effect

This is a demo of a Line chart using the trace() effect and with no vertical gridlines. The chart uses the CSV reader that's bundled with RGraph and the CSV data is part of the page - in a DIV tag. The tooltips are significantly customised and the range of the axes is smaller than normal by making use of the yaxisScaleMin and yaxisScaleMax properties.
