A Line chart using imported table data

[No canvas support]

This is a simple example of a chart that also use the HTMLTable utility.

This goes in the documents header: 
<script src="RGraph.common.core.js"></script>
<script src="RGraph.common.table.js"></script>
<script src="RGraph.bar.js"></script>
Put this where you want the chart to show up: 
<div style="padding: 15px">
    <canvas id="cvs" width="750" height="300">[No canvas support]</canvas>
</div>
