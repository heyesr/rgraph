An SVG Horizontal Bar chart showing property prices

A Horizontal Bar chart that's showing property prices using the grow() effect. The tooltips are customised using the default CSS values (which are applied to the tootlips after they've been created).

This goes in the documents header: 
<script src="RGraph.svg.common.core.js"></script>
<script src="RGraph.svg.hbar.js"></script>
Put this where you want the chart to show up: 
<div style="padding: 15px">
    <div style="width: 700px; height: 500px" id="chart-container"></div>
</div>
