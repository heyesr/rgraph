A demo of the Chrome-only dotted or dashed background grid

This uses the newer addition to canvas from the canvas v5 specification - native dotted or dashed lines. As browsers add support you will be able to use the option in those too but initially it's only supported in Chrome. 

obj.Set('backgroundGridDashed', true);
obj.Set('backgroundGridDotted', true);

<script src="RGraph.common.core.js"></script>
