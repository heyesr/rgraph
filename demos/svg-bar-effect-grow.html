An SVG Bar chart using the grow() effect

A simple red Bar chart that uses the grow() effect, has a title and labels too. Not too much to the chart - it's quite a basic bar chart that uses the grow() effect.

This goes in the documents header: 
<script src="RGraph.svg.common.core.js"></script>
<script src="RGraph.svg.bar.js"></script>
Put this where you want the chart to show up: 
<div style="padding: 15px">
